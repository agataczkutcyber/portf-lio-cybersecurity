Gestão de Riscos ISO 27005
Projeto de Gestão de Riscos desenvolvidos para estudo de GRC

Objetivo:
Identificar, analisar e tratar riscos em um ambiente simulado de e-commerce

Atividades:
- Inventário de ativos
- Identificação de ameaças
- Identificação de vulnerabilidades
- Avaliação de riscos
- Plano de tratamento
- Matriz de riscos

Frameworks Utilizados:
- ISO 27001
- ISO 27005
- NIST CSF
