Homelab SOC com Wazuh

Objetivo:

Construir um ambiente de monitoramento de segurança para estudo do SOC e análise de eventos

Ambiente:

- Ubuntu Server
- Wazuh Manager
- Agentes Linux
- Kali Linux
- VirtualBox

Atividades Realizadas

- Instalação de Wazuh
- Configuração de agentes
- Monitoramento de logs
- Detecção de eventos de segurança
- Análise de alertas

Tecnologias:

- Wazuh
- Linux
- VirtualBox
- TCP/IP
- Logs de Segurança
